First launch: because the app is signed locally (not notarized), right-click it in Applications → Open → Open the first time. If macOS still blocks it, go to System Settings → Privacy & Security → Open Anyway. Run it from your Applications folder (not from the .dmg window or Downloads) so it can keep itself up to date — if you open it from the wrong place it will offer to move itself there for you.
